## 📌 Project Overview

The **HealthCare Risk Prediction App** is a machine learning-based web
application that predicts a patient's health risk based on input
parameters.

It uses an **XGBoost model** to provide accurate predictions and is
deployed using **Streamlit** for an interactive UI.

---

## 🎯 Features

- 📊 Predicts health risk instantly\
- 🧠 Uses trained XGBoost model\
- ⚡ Fast and user-friendly interface\
- 🌐 Web-based (no installation needed)

---

## 🛠 Tech Stack

- **Frontend:** Streamlit\
- **Backend:** Python\
- **ML Model:** XGBoost\
- **Libraries:**
  - pandas\
  - numpy\
  - scikit-learn\
  - xgboost

## ▶️ Run Locally

### 1. Clone the repository

git clone https://github.com/K-Parth/HealthCare-Risk-Prediction-App.git\
cd HealthCare-Risk-Prediction-App

### 2. Install dependencies

pip install -r requirements.txt

### 3. Run the app

streamlit run app.py
